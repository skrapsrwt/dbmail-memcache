DBMAIL-HTTPD(8)
===============


NAME
----
dbmail-httpd - speaks the HTTP-1.1 protocol

SYNOPSIS
--------
dbmail-httpd [-f configfile] [-p pidfile] [-nvVh]

DESCRIPTION
-----------
Exposes DBMail's datamodel through a REST-full api. 

Please refer to contrib/dbmailclient.php for supported
url paths.
